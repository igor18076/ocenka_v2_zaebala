# Checkbox

Checkbox (default) or radio (`type="radio"`). Used in the ФСО checklist, filters and forms.

```jsx
<Checkbox label="Заполнено задание на оценку" defaultChecked />
<Checkbox label="Проведен анализ рынка" />
<Checkbox type="radio" name="approach" label="Сравнительный подход" defaultChecked />
```

Forwards native input attributes (`checked`, `onChange`, `name`, `disabled`).
