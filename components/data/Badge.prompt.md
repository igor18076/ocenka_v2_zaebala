# Badge

Compact label chip for categories, counts and tags. Tones map to the feedback palette.

```jsx
<Badge tone="success" pill dot>Высокая сопоставимость</Badge>
<Badge tone="warning" pill>Требует проверки</Badge>
<Badge tone="brand">DOCX</Badge>
<Badge tone="outline">12 документов</Badge>
```

Tones: `neutral · brand · success · warning · danger · info · outline`. Props: `pill`, `dot`. For request lifecycle states use **StatusBadge**.
