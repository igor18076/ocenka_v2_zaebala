# IconButton

Icon-only button for toolbars, topbar controls and table row actions. Always pass `aria-label`.

```jsx
<IconButton aria-label="Уведомления"><Bell size={18} /></IconButton>
<IconButton aria-label="Ещё" bordered><MoreHorizontal size={18} /></IconButton>
<IconButton aria-label="Закрыть" size="sm"><X size={16} /></IconButton>
```

Sizes: `sm · md`. Props: `bordered` (adds border + card bg), all native button attributes.
